O objetivo principal do projeto é desenvolver um protótipo acadêmico demonstrativo de monitoramento energético baseado em IoT e Cloud, utilizando Python para simulação, armazenamento e análise de dados. Já os objetivos secundário são:
# Simular sensores IoT para gerar dados de consumo da energia
# Armazenar as informações recebidas em um banco de dados
# Fazer um dashboard em Python para ver o consumo
# Documentar todas as etapas
